
Overview
~~~~~~~~

    The content of the 
    
        ./converted_to_HTML
    
    has been generated from the content of the
    
        ./in_vimwiki_format
    
    by using vimwiki command ":VimwikiAll2HTML". 
    The content of the 
    
        ./converted_to_HTML
    
    is meant to be browsed with terminal based web browsers like
    "links" or "elinks" or "lynx". Browser invocation example:
    
        elinks ./converted_to_HTML/index.html
