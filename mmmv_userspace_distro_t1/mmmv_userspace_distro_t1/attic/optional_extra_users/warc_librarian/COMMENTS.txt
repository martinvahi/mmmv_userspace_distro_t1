
The warc_librarian is a user that is meant for
creating WARC (Web Archive) files

    https://www.loc.gov/preservation/digital/formats/fdd/fdd000236.shtml
    (archival copy: https://archive.is/sOIKi )

WARC files can be viewed by running a WARC 
displaying server that runs as a specialized http proxy. 
The proxy server might be used by using the 

    http://www.midori-browser.org/

Example Bash aliases for running the server 
and for creating the WARC files can be found from 

    ./_bashrc

which is an example of the 

    ~/.bashrc
