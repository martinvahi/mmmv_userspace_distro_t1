Manual Page Look-Up

Internal links within the HTML version of the REDUCE manual should not be used externally because they may change when the manual is rebuilt from source, which typically happens a few times a year. Instead, use a URL of the form ?query string, where query string is any (case-sensitive) string of characters that appears uniquely anchored at the start of the title of an entry in the REDUCE manual table of contents. (Don't use section numbers, which may change.) For example, these links should reliably access appropriate sections of the manual describing User Contributed Packages:

  • ?EXCALC:
  • ?GROEBNER:
  • ?LINALG:
  • ?ODESOLVE:
  • ?REDLOG:
  • ?SPECFN:

Alternatively, for interactive experimentation, enter a query string in the field below and hit Enter or click on the Go button.

If a query string is not found then the REDUCE manual table of contents is displayed.
