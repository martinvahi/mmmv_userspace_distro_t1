20.48 REDLOG: Extend REDUCE to a Computer Logic System

The name REDLOG stand for REDuce LOGic system. Redlog implements symbolic algorithms on ﬁrst-order formulas with respect to user-chosen ﬁrst-order languages and theories. The available domains include real numbers, integers, complex numbers, p-adic numbers, quantiﬁed propositional calculus, term algebras.

Documentation for this package can be found online.

Authors: Andreas Dolzmann and Thomas Sturm
