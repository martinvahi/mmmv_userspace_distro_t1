18.1 Introduction

To access the prettyprinter described in this chapter you must load the module PP.

A prettyprinter takes as input a stream of characters and prints them with aethetically appropriate indentations and line breaks. For example

(if (vectorp data) (vector-size data) (if (stringp data)  
    (string-length data) (length data)))

(if (vectorp data)  
  (vector-size data)  
  (if (stringp data)  
    (string-length data)  
    (length data)))
