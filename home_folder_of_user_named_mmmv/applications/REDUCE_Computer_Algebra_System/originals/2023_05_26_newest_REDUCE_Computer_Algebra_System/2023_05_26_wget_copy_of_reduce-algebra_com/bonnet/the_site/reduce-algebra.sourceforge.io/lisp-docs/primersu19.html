4.3 Generic arithmetic with domain elements

There is a complete set of arithmetic functions for performing arithmetic operations with domain elements:

!:zerop(a)test a = 0
!:plus(a,b)a + b
!:difference(a,b)a - b
!:minus(a)-a
!:minusp(a)test a < 0
!:times(a,b)a * b
!:quotient(a,b)a∕b
!:recip(a)1∕b
!:expt(a,n)an
!:gcd(a,b)greatestcommondivisorof (a,b)
