4 Lists

A list is an object consisting of a sequence of other objects (including lists themselves), separated by commas and surrounded by braces. Examples of lists are: 

        {a,b,c}

        {1,a-b,c=d}

        {{a},{{b,c},d},e}.

The empty list is represented as 

        {}.
