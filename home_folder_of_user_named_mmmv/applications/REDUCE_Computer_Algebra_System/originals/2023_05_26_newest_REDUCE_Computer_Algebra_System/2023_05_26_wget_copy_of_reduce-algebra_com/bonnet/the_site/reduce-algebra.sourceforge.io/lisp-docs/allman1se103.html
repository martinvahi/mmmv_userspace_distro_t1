26.8 Disassembler

For most implementations of PSL, a disassembler for compiled functions is available which gives insight into the ’real’ code. It is a tool for code optimization only and not meant to give insight for the ’normal’ LISP user.
Example:

(load disassemble)  
NIL  
(disassemble 'get)
