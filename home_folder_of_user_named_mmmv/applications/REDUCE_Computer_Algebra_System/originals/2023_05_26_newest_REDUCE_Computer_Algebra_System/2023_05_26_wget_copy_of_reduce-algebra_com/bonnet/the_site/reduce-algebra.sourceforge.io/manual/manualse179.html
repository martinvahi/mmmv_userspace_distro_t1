Authors: Thomas Wolf and Eberhard Schruefer

20.56.1 Overview

A detailed description is available through the online tutorial
https://lie.ac.brocku.ca/crack/susy/. An essentially equivalent description is available after loading SSTOOLS and issuing the command sshelp()$. The correct functioning of all procedures is tested through reading in and running sstools.tst. This test also illustrates the commutator rules for products of the diﬀerent ﬁelds and their derivatives with respect to besonic and fermionic variables.
