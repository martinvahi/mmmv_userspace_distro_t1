7.13 The Pochhammer Notation

The Pochhammer notation \((a)_k\) (also called Pochhammer’s symbol) is supported by the binary operator Pochhammer(a,k). For a non-negative integer \(k\), it is deﬁned as (http://dlmf.nist.gov/5.2.iii) \begin {align*} (a)_0 &= 1, \\ (a)_k &= a(a+1)(a+2)\cdots (a+k-1). \end {align*}

For \(a \neq 0, -1, -2, -3, \ldots \), this is equivalent to \[ (a)_k = \frac {\Gamma (a+k)}{\Gamma (a)}. \] When \(n\) is integral, the deﬁning product is expanded (assuming the switch exp is on). With rounded oﬀ, this expression is evaluated numerically if \(a\) is numerical and \(k\) is integral, and otherwise may be simpliﬁed where appropriate. The simpliﬁcation rules are based upon algorithms supplied by Wolfram Koepf [Koe92].

The Pochhammer symbol is used quite extensively in the simpliﬁcation and numerical evaluation of special functions.
