12 File Handling Commands

In many applications, it is desirable to load previously prepared REDUCE ﬁles into the system, or to write output on other ﬁles. REDUCE oﬀers four commands for this purpose, namely, in, out, shut, load, and load_package. The ﬁrst three operators are described here; load and load_package are discussed in Section 23.2.
