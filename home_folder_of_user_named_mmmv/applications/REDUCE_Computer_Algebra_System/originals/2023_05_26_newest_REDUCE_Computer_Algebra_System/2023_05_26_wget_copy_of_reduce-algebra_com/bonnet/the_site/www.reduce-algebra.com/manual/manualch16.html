16 Series Expansion

Expanding an algebraic expression into a series can be done by standard REDUCE operators, namely df, sub, and possibly limit. Nevertheless, there are many cases where this straightforward method fails. REDUCE oﬀers two diﬀerent operators for this purpose:

taylor

computes a truncated power series.

ps

computes extendible power series.

fps

computes formal power series.
